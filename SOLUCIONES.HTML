Soluciones para los problemas del computador

Es la resolución de un problema o error ya sea de hardware o de software en un dispositivo electrónico como un computador a continuación le encontrara información para realizar un diagnostico exitoso

Soluciones para la memoria RAM

1.comprobar si los slots no se encuentren obstruido
2.Limpiar los slots en caso de que los errores persistan es necesario cambiar el RAM

Solución para el del disco duro

1.comprobar si los slots no se encuentren obstruido
2.Limpiar los slots en caso de que los errores persistan es necesario cambiar el disco duro

sistema operativo (malware) o error de actualizacion

Malware

1.Consultar con su respectivo antivirus en caso de no ser detectado proceda a consultar con otro antivirus usar un fillware
2.En caso de lo anterior no funcione entonces es necesario realizar un restablecimiento

solución del error al actualizar el computador

1.Regresar a la versión anterior
2.Reiniciar el equipo
3.Comprobar conexion a internet

Soluciones para la Fuente de poder

1.Comprobar si esta bien conectada
2.Comprobar si los enchufes producen la suficiente energía
3.En caso de lo anterior no funcionar entonces es necesario cambiar la fuente de poder
