Zócalo del procesador

Los zócalos del procesador (también conocidos como socket en inglés) son conectores en la placa base (motherboard) de una computadora donde se instala el procesador (CPU). Estos zócalos permiten que la CPU se conecte físicamente y pueda comunicarse con otros componentes del sistema

Tipos de zócalos

Existen varios tipos de zócalos, como LGA (Land Grid Array) para procesadores Intel, y PGA (Pin Grid Array) para algunos procesadores AMD. Cada uno tiene características específicas que determinan la compatibilidad con diferentes modelos de CPU

Bancos de memoria (RAM slots)

Los bancos de memoria, o slots de RAM, son las ranuras en la placa base de una computadora donde se instalan los módulos de memoria RAM.

Número de Slots:La mayoría de las placas base tienen entre 2 y 8 slots de RAM, aunque algunas pueden tener más. El número de slots influye en la cantidad máxima de memoria que se puede instalar.

Dual Channel y Multi-Channel:: Para obtener un mejor rendimiento, se recomienda instalar la RAM en pares, aprovechando configuraciones de dual channel (dos módulos) o quad channel (cuatro módulos). Esto permite una mayor eficiencia en la transferencia de datos

Overclocking: Algunas placas base y módulos de RAM soportan lo que permite aumentar la velocidad de la memoria más allá de las especificaciones estándar, pero esto puede requerir ajustes en la configuración del BIOS.

Condensadores y reguladores de voltaje

Condensadores

Un condensador (también llamado capacitor) es un componente electrónico pasivo que almacena energía en forma de carga eléctrica. Se utiliza para diversos fines en los circuitos electrónicos, como filtrar ruidos eléctricos, suavizar fluctuaciones de voltaje, almacenar energía y eliminar picos de voltaje.

regualador de voltaje

Un regulador de voltaje es un dispositivo electrónico que mantiene un voltaje constante en un circuito, independientemente de las variaciones en la entrada o en la carga. Los reguladores de voltaje son fundamentales para proporcionar una salida de voltaje estable, lo cual es esencial en dispositivos electrónicos sensibles, como computadoras, teléfonos móviles y sistemas de alimentación Existen dos tipos principales de reguladores de voltaje:
a.Reguladores Lineales
b.Reguladores Switching (Conmutados)
